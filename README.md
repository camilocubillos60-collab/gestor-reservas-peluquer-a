# Gestor de Reservas para Peluquería

Proyecto desarrollado en Java como práctica de programación orientada a objetos básica.

## Funcionalidades

- Agendar citas.
- Listar reservas.
- Cancelar reservas.
- Reporte del día.
- Validación de datos.

## Tecnologías

- Java
- Scanner
- Arreglos
- Métodos
- Estructuras de control
